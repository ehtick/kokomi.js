    Class ThinFilmFresnelMap

    Credit: https://github.com/DerSchmale/threejs-thin-film-iridescence

    Demo: https://kokomi-playground.vercel.app/entries/#liquidCrystal

    Classdesc

    ThinFilmFresnelMap is a lookup texture containing the reflection colour. The texture index value is dot(normal, view). The texture values are stored in approximated gamma space (power 2.0), so the sampled value needs to be multiplied with itself before use. The sampled value should replace the fresnel factor in a PBR material.

    Param: filmThickness

    The thickness of the thin film layer in nanometers. Defaults to 380.

    Param: refractiveIndexFilm

    The refractive index of the thin film. Defaults to 2.

    Param: refractiveIndexBase

    The refractive index of the material under the film. Defaults to 3.

    Param: size

    The width of the texture. Defaults to 64.

    Author

    David Lenaerts http://www.derschmale.com
